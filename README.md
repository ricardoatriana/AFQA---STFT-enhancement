# AFQA---STFT-enhancement

Enhanced the AFQA (Automated Fingerprint Quality Assessment) Toolbox by integrating a Short-Time Fourier Transform (STFT) fingerprint enhancement module into the biometric processing pipeline.

Conducted image processing, feature extraction, and machine learning experimentation using Random Forest models to evaluate fingerprint quality assessment performance.

Compared STFT-based enhancement methods against Oriented Gabor Enhancement techniques to analyze quality assessment accuracy and model generalization.
Performed statistical analysis and operational testing on fingerprint datasets to evaluate enhancement effectiveness and quality score distributions.
